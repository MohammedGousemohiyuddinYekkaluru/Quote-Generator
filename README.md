# Quote Generator

A simple web app that fetches and displays random quotes with a loading state between quotes using HTML, CSS & JavaScript.

## Files

- `index.html` — main markup  
- `style.css` — UI styling  
- `script.js` — logic for fetching quotes and handling the loading state  

## Usage

1. Open `index.html` in your browser.  
2. Click the “Refresh” button to load a random quote.  
3. A loading animation appears while fetching the next quote.
